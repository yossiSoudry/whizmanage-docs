## מבוא למערכת הייבוא

<Note title="גרסה ראשונית" type="info">
כרגע מערכת הייבוא תומכת רק באפשרות ייבוא באמצעות Google Sheets. אפשרויות נוספות כמו ייבוא מקבצי WooCommerce ו-WhizManage יתווספו בגרסאות הבאות.
</Note>

מערכת הייבוא של WhizManage מאפשרת לכם לייבא מוצרים בקלות ממגוון מקורות, לעדכן מוצרים קיימים, ולנהל את הליך הייבוא באופן מבוקר. המערכת תומכת במספר פורמטים וממשקים, כדי לאפשר לכם לייבא ולנהל את מוצריכם באופן הטוב ביותר.

### יתרונות מערכת הייבוא

1. **גמישות בייבוא**
   - ייבוא ישירות מ-Google Sheets
   - גישה נוחה למערכת מכל מקום
   - ניהול מוצרים בזמן אמת

2. **בקרה מלאה**
   - סנכרון דו-כיווני בין המערכת לקובץ
   - עדכון נתוני מוצרים בממשק מוכר
   - שיתוף פעולה קל עם צוות העבודה

3. **שילוב נתונים חכם**
   - ייבוא תמונות מוצרים אוטומטי
   - שיוך קטגוריות אוטומטי
   - טיפול אוטומטי בווריאציות מוצרים

### תפיסת העבודה המומלצת

1. **תכנון מראש**
  - הכינו את הנתונים בפורמט המתאים
  - השתמשו בקובץ הדוגמה כתבנית
  - בצעו ייבוא ניסיוני בסביבת בדיקות

2. **שימוש בתכונות מתקדמות**
  - נצלו את אפשרויות העבודה השיתופית ב-Google Sheets
  - השתמשו בנוסחאות גיליון אלקטרוני לחישובים אוטומטיים
  - הגדירו תנאי עיצוב לזיהוי בעיות בנתונים

3. **בקרת איכות**
  - בדקו את המוצרים לאחר הסנכרון
  - וודאו שכל הנתונים נכנסו כראוי
  - שמרו על גיבוי עדכני של נתוני המוצרים

<Note title="חשוב" type="warning">
למידע מפורט על תהליך הייבוא מ-Google Sheets, כולל הוראות צעד-אחר-צעד, ראו את הדף <a href="/he/docs/import/spreadsheet">ייבוא מ-Google Sheets</a>.
</Note>
