## Managing Existing Variations

<Step>
<StepItem title="Variations Table">

The variations table displays all existing variations for the product:

| Column | Description | Example |
|--------|------------|---------|
| **Attributes** | Combination of variation attributes | Red / M |
| **Regular Price** | Regular price of the variation | $99.90 |
| **Sale Price** | Sale price if exists | $79.90 |
| **Description** | Variation-specific description | |
| **Menu Order** | Display order in shop | 1 |

<img src="https://placehold.co/600x400?text=Variations+table" alt="Variations table" />
</StepItem>

<StepItem title="Editing Variations">

For each variation, you can edit:

1. **Prices**
  - Regular price
  - Sale price

2. **Additional Information**
  - Variation description
  - Display order in shop

<Note title="Important" type="info">
Changes are only saved after clicking the "Save Changes" button.
</Note>
</StepItem>

<StepItem title="Deleting Variations">

You can delete variations that are no longer needed:
1. The variation will be removed from the variations list
2. Deleted variations cannot be recovered
3. Deletion takes effect only after saving changes

<Note title="Warning" type="warning">
Make sure you really want to delete the variation before confirming deletion.
</Note>
</StepItem>
</Step>
