## Installing the WhizManage Plugin

The WhizManage plugin is installed similarly to any other WordPress plugin. Below is a simple guide for installing the system and activating it.

<Step>
<StepItem title="Downloading the Plugin">

There are two ways to download and install the WhizManage plugin:

**Option 1: Install directly from the WordPress admin interface**
1. Log in to your WordPress admin area
2. Go to "Plugins" > "Add New"
3. In the search box, type "WhizManage"
4. When the plugin appears in the list, click "Install Now"

**Option 2: Download from the WhizManage website and install manually**
1. Visit the [WhizManage](https://whizmanage.com) website
2. Go to the downloads page and download the plugin ZIP file
3. In your WordPress admin, go to "Plugins" > "Add New"
4. Click "Upload Plugin" at the top of the screen
5. Select the downloaded ZIP file and click "Install Now"

<img src="https://placehold.co/800x300?text=WhizManage+Plugin+Installation" alt="WhizManage Plugin Installation" />
</StepItem>

<StepItem title="Activating the Plugin">

1. After installation, click "Activate Plugin"
2. The WhizManage plugin will be added to the WordPress sidebar menu
3. Click on "WhizManage" in the sidebar menu to enter the plugin management interface

<Note title="Note" type="info">
After the initial activation, the WhizManage plugin is available in the basic (free) version with certain limitations on functionality.
</Note>
</StepItem>

<StepItem title="Entering License (for Pro Version Users)">

If you've purchased one of the Pro versions of WhizManage:

1. Make sure you've received the license code by email after purchase
   (You can also find the license in your personal area on the [WhizManage](https://whizmanage.com) website)
2. In the WhizManage management interface, click on your user image in the top corner
3. In the dropdown menu, select "License"
4. Enter the license code you received in the appropriate field
5. Click "Activate License"
6. You will receive a confirmation message that the license has been successfully activated

<img src="https://placehold.co/800x300?text=WhizManage+License+Activation" alt="WhizManage License Activation" />

<Note title="Important" type="warning">
Make sure you enter the license code exactly as received, including dashes and upper/lowercase letters.
</Note>
</StepItem>
</Step>

## Version Updates

When an update is available for the plugin:

1. You will receive a notification in the WhizManage system (as shown in the image)
2. Click "Update now" to upgrade the plugin
3. After the upgrade, the system will automatically update to the new version

<img src="https://placehold.co/800x300?text=WhizManage+Update+Notification" alt="WhizManage Update Notification" />

<Note title="Important" type="warning">
It's recommended to backup your site before updating the plugin, especially for major updates.
</Note>

## System Requirements

For optimal operation of WhizManage:

- Updated WordPress version
- Updated WooCommerce version
- Modern browser (Chrome, Firefox, Edge, Safari)
- Stable internet connection

<Note title="Tip" type="info">
For optimal performance, it's recommended to use the latest versions of WordPress and WooCommerce.
</Note>
