---
title: Lists
description: This section provides an overview of creating and using lists in the Documents boilerplate.
---

Learn how to create and render different types of lists in the Documents boilerplate. Examples include checklists, ordered lists, unordered lists and nested lists.

## Checklist

- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media

```jsx
- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
```

## Simple List

- Item 1
- Item 2
- Item 3

```jsx
- Item 1
- Item 2
- Item 3
```

## Number List

1. Research
2. Draft the content
3. Review and edit
4. Publish

```jsx
1. Research
2. Draft the content
3. Review and edit
4. Publish
```

## Nested List

- Main Category 1
  - Sub Item 1.1
  - Sub Item 1.2
- Main Category 2
  - Sub Item 2.1
  - Sub Item 2.2

```jsx
- Main Category 1
  - Sub Item 1.1
  - Sub Item 1.2
- Main Category 2
  - Sub Item 2.1
  - Sub Item 2.2
```

Use these examples as a foundation to create and customize lists that fit your project's needs.
